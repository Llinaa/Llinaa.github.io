发表于 2018-09-26 更新于 2020-05-30 分类于 CSS
  • 三种技术：弹性布局、弹性图片、媒体查询
  • 浏览器支持：先写一个较轻量的代码架构，然后根据所需的体验针对能力更强的浏览器进行扩展，包括视觉和功能（渐进增强）
  • 浏览器特性查询：https://caniuse.com
